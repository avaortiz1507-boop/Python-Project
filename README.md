# In this python project, I have created multiple Python programs.

## Greeter
In the python greeter.py, the user is prompted to type their name.
---

## Calculator
In the calculator.py, the user is prompted to add, subtract, multiply or divid 2 numbers.
---

## Menu
In the menu.py, the user is prompted to choose 4 options: particpate in survey, compare 2 numbers, answer a question or exit the survey.
---

## FizzBuzz
In the fizz_buzz.py, the user is prompted to enter a number inside of a range. If the number is divisible by 3, 5 or both, it will print a certain word.
---

## Vowel Counter
In the counting_vowels.py, the user is prompted to enter a word that has vowels. Python will then count and display how many vowels the word has. 
---

## Guess the Number
In the number_guessing_game.py, the user is asked to guess the number Python is "thinking".

## Text Processing
In the word_counting.py, the user is asked what their favorite season is and why. Then, the number of words and the average length of all the words are calculated and displayed.
